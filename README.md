# 📇 名刺スキャナー

スマホで名刺を撮影するだけで、Gemini AIが自動的に情報を抽出してスプレッドシート用に出力するWebアプリです。

## ✨ 機能

- 📸 スマホカメラで名刺を撮影
- 🤖 Gemini AIによる自動データ抽出
  - 会社名
  - 氏名
  - 役職
  - 電話番号
  - メールアドレス
  - 住所
  - その他の情報
- ✏️ 抽出結果の確認・編集
- 📥 CSV形式で出力（Excelやスプレッドシートに貼り付け可能）
- 📚 スキャン履歴の保存（最大50件）
- 📱 スマホ完全対応のレスポンシブデザイン

## 🚀 セットアップ

### 1. Gemini APIキーの取得

1. [Google AI Studio](https://makersuite.google.com/app/apikey) にアクセス
2. Googleアカウントでログイン
3. 「Create API Key」をクリック
4. APIキーをコピー

### 2. アプリの起動

#### 方法1: ローカルサーバーで起動（推奨）

```bash
# Python 3がインストールされている場合
cd business-card-scanner
python -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```

#### 方法2: 直接HTMLファイルを開く

1. `index.html` をダブルクリック
2. ブラウザで開く（一部の機能が制限される可能性があります）

### 3. APIキーの設定

1. アプリ右上の⚙️ボタンをクリック
2. 取得したGemini APIキーを入力
3. 「保存」をクリック

## 📱 使い方

1. **名刺を撮影**
   - 「📷 名刺を撮影」ボタンをタップ
   - スマホのカメラで名刺を撮影
   - 明るい場所で、名刺全体が写るように撮影

2. **自動解析**
   - 撮影後、自動的にAIが名刺情報を解析
   - 数秒で結果が表示されます

3. **結果の確認・編集**
   - 抽出された情報を確認
   - 必要に応じて修正・追加

4. **エクスポート**
   - **CSV出力**: スプレッドシートに貼り付け可能なCSVファイルをダウンロード
   - **Sheetsに追加**: (準備中) Google Sheetsに直接追加

5. **履歴から再利用**
   - 過去にスキャンした名刺は「スキャン履歴」に保存
   - タップするだけで情報を呼び出せます

## 📊 CSVをスプレッドシートに取り込む方法

### Google スプレッドシート

1. Google スプレッドシートを開く
2. 「ファイル」→「インポート」
3. 「アップロード」タブからダウンロードしたCSVを選択
4. インポート場所を選択して「データをインポート」

### Microsoft Excel

1. Excelを開く
2. 「データ」タブ→「テキストまたはCSVから」
3. ダウンロードしたCSVファイルを選択
4. 文字コードを「UTF-8」に設定して読み込み

## 🔐 プライバシーとセキュリティ

- **データの保存**: すべてのデータはブラウザのローカルストレージに保存されます
- **外部送信**: 名刺画像はGemini APIに送信されますが、48時間後に自動削除されます
- **APIキー**: APIキーはブラウザのローカルストレージに保存されます（暗号化されていません）

## 🛠️ トラブルシューティング

### カメラが起動しない

- ブラウザにカメラの許可を与えてください
- HTTPSまたはlocalhostで起動してください

### 解析が失敗する

- APIキーが正しく設定されているか確認
- 名刺が明るくはっきり写っているか確認
- インターネット接続を確認

### 日本語が文字化けする

- CSVをUTF-8形式で開くようにしてください
- Excelの場合、インポート時に「UTF-8」を選択

## 🙋 よくある質問

### Q: 無料で使えますか？

A: はい。Gemini APIは無料枠があり、個人利用なら十分です。

### Q: オフラインで使えますか？

A: 現在は名刺解析にインターネット接続が必要です。将来的にPWA対応を予定しています。

### Q: どんな名刺でも読み取れますか？

A: 日本語・英語の名刺に対応しています。手書きやデザイン性の高い名刺は精度が下がる場合があります。

### Q: データはどこに保存されますか？

A: すべてブラウザのローカルストレージに保存され、外部サーバーには保存されません。

## Google Sheets 直接書き込み

1. 右上の設定を開く
2. 「Googleで接続」で認証
3. スプレッドシートのURLまたはIDを入力
4. 「シート一覧を取得」で書き込み先シートを選択
5. 抽出結果の「シートに追加」または「まとめてシートに追加」を押す

※ 認証にはOAuthクライアントIDが必要です。`app.js` の `clientId` を自分のプロジェクトのものに変更してください。
※ HTTPS もしくは `http://localhost` で起動してください。
